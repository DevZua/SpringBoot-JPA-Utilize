## JPA shop 웹 화면 구성하기
1. 회원 기능
    * 회원 등록, 조회
2. 상품 기능
    * 상품 등록, 수정, 조회
3. 주문 기능
    * 상품 주문
    * 주문 내역 조회
    * 주문 취소
4. 기타 요구 사항
    * 상품은 재고 관리가 필요하다.
    * 상품의 종류는 반드시 도서, 음반, 영화가 있다.
    * 상품을 카테고리로 구분할 수 있다.
    * 상품 주문 시 배송정보를 입력할 수 있다.
